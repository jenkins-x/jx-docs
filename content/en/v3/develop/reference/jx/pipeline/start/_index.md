## jx pipeline start

Starts one or more pipelines

***Aliases**: build,run*

### Usage

```
jx pipeline start
```

### Synopsis

Starts the pipeline build.

### Examples

  ```bash
  # Start a pipeline
  jx pipeline start foo
  
  # Select the pipeline to start
  jx pipeline start
  
  # Select the pipeline to start and tail the log
  jx pipeline start -t

  ```
