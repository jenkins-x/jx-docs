加速您在 Kubernetes 上的持续交付
Jenkins X 提供了管道自动化、内置 GitOps 和预览环境，以帮助团队在任何规模上进行协作并加速软件交付速率。

自动化 CI/CD

基于 GitOps 的 Tekton 流水线

不需要对 Kubernetes 、容器或 Tekton 有深入的了解，Jenkins X 会为您的项目自动生成出色的Tekton管道，你可以 通过 GitOps 管理完全实现CI和CD

环境

通过 GitOps 进行环境更新

每个团队都有一组环境。 Jenkins X 通过 GitOps 自动管理环境并在环境之间更新新版本的应用程序


预览环境

Pull Request 的预览环境

Jenkins X 为您的 Pull Request 自动启动预览环境，这样你就可以在修改合并到主分支之前快速获得反馈。

ChatOps

对 Issue 和 Pull Request 的反馈和聊天

Jenkins X 自动评论您的提交、问题和拉取请求，并在代码准备好预览、升级到环境或自动生成拉取请求以升级版本时提供反馈
