## jx get

Display one or many resources

### Synopsis

Display one or many resources. 

Valid resource types include: 

  * environments (aka 'env')  
  * pipelines (aka 'pipe')  
  * urls (aka 'url')

```
jx get TYPE [flags]
```

### Examples

```
  # List all pipeines
  jx get pipeline
  
  # List all URLs for services in the current namespace
  jx get url
```
